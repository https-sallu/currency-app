# 💱 Dynamic Currency Converter

A responsive, real-time currency converter web application built with React and Vite. This project combines modern front-end development practices with real-time financial data, featuring a custom-built API hook and a polished UI styled with Tailwind CSS.

## 🚀 Features
* **Real-Time Exchange Rates:** Fetches live, up-to-date currency data using the `@fawazahmed0/currency-api` via an NPM-hosted CDN.
* **Custom React Hooks:** Utilizes a dedicated `useCurrencyInfo` hook to abstract API calls and keep component logic clean.
* **Interactive UI:** Includes a quick-swap functionality to instantly reverse the "From" and "To" currency states.
* **Modern Styling:** Fully responsive glass-morphism design built with Tailwind CSS, set against a professional financial chart background.
* **Robust State Management:** Efficiently handles bidirectional state updates and prop passing across modular `InputBox` components.

## 🛠️ Tech Stack
* **Frontend:** React.js, Tailwind CSS
* **Build Tool:** Vite
* **API:** Free Currency API
